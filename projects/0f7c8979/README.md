## カスタマイズ方法

### 配色変更
`style.css` の `:root` セクションでCSS変数を変更してください。

```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    /* ... */
}
```

### コンテンツ変更
`index.html` の各セクション内のテキストを編集してください。

### 機能追加
`script.js` に新しい機能を追加できます。
